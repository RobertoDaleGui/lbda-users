# PASSOS PARA CONFIGURAR

- Editar o nome do projeto no ```serverless.yml```
- Editar o nome do projeto no ```package.json```
- ```yarn install```

# Script pre-commit e demais hooks

- Copie os arquivos da pasta hook para ```.git/hooks``` dentro da pasta do projeto e então renomeie o arquivo sem a extenção.
